Hyper Text Mark-up Language (from here on known as HTML) is one of many coding languages that can be used to create websites. HTML5 is the newest version so far of HTML which has more features like new elements, attributes and behaviours. HTML is basically the skeleton of the website and is used to set the layout of the website and put the content of the page on to the website so that other languages could be used to enhance the website to looking more professional. HTML code mainly uses tags to show the start and the end of a line of code which represents which object the code applies to. HTML coding language can also have other languages in it for example in HTML CSS can be used however its layout is different from that of CSS. HTML does not allow the user to use all the coding from CSS however contains the basics making it easy for the user to transfer from one language to another.

This example of the structure is the basic structure to the start of creating a website. With every tag, shown by the two angle brackets, there should be an opening and a closing tag. With the first tag “<html>” it tells the software that they are working with HTML.  In the HTML tag there will always be a heading and a body. Within the heading there should be a title, this is where the tab title will go and what the user will see on the tabs. It will also contain any references to other files which may have other coding languages so that the two files can be linked allowing the user to have more than one coding language being used in the website. Within the body there will be all of the website’s content. This will be created with multiple different codes like using text tags, div tags, table tags etc. which will then create the content and layout of the page however this can be overwritten with the CSS file.

This example is a container that is in the website that I have created called Katie’s Café which contains an two images and a menu navigation bar. Specifically the example shows the container which is between the two angle brackets and has “div” between them. In HTML it is important to note that when a tag is opened it must be closed at some point of the document. This tag is to create a visible or invisible box which other elements can be put in. This tag can also be used a shape rather than a container, for example the user could use some CSS by colouring and shaping the div tag with round corners and make it the background of a button. Div tags can have many different uses. In the example I have shown there is are references to other files which contain other coding languages so that the div tag can be customised to the user's need.

This example is showing the use of text tags and the different types that has been used. In this example there is use of heading tags, paragraph tags and break tags. The first tag in the example is the heading tag, which is used to as it will default the font size to be larger than the paragraph text. These headings can be split into sub-headings as well. The tag for a title heading would be “<h1>Example</h1>” and then the higher the number the more sub-headings it would be. The next line uses the paragraph tag which is shown by the tag “<p>paragraphs</p>” which creates a line of text. To create a paragraph where the user has the text stacked above each other the user must use a break tag. This puts the next line of text on a separate line.
