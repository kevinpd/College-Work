HTML

HTML (HyperText Markup Language) is a programming language used in the creation of websites. The initial release of HTML was in 1993 and the idea was proposed by Tim Berners-Lee. HTML is the base of a website so when adding other coding languages it is important to link the files or embed the code. The three programming languages, CSS3 and JavaScript, are the basis of most modern websites. There are many programs that support the use of HTML the simplest of being Notepad++. There have been many different versions of HTML every newer one with better features and easier use, the latest HTML version being HTML5. When it comes to HTML and many other different programming languages has a certain structure that needs to be followed.

CSS (Cascading Style Sheets) is a programming language that is used with HTML however since HTML is used to create the basis of the website. Instead CSS is used to create the presentation of the website. For example CSS is used to change the layout, colours, font etc. CSS is a programming language that was initially released in 1996 not long after HTML. With CSS it can be both embedded in the HTML file or create a new file and link it with the HTML. CSS like many other programming languages has a certain format which needs to be followed. CSS has some interactivity with the user however not much, which is why the use of JavaScript is also benefitial to creating a successful website.
