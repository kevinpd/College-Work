There are two different methods that can be used to access external CSS, the first that I will be going through is the link method. The link method looks very similar to the import method however the link method allows the user to link the file with a ‘href’ tag inside a link tag rather than using the ‘@import’ tag which is in the style tag. When it comes to linking the external the user is able to cascade the style sheets. This is when the user uses a link inside the external CSS file to create something that looks like it is cascading. This can be very frustrating to work with if there is anything that the user will want to change or will have to troubleshoot. This can also be bad if the user has made an error or there is a problem with the style sheet as this would corrupt the following CSS files.

 

The link method is a fairly new method that allows the user to reference the external CSS file rather than importing the file directly into the code. There are many upsides to linking the external CSS file rather than importing it. The biggest one is that by linking the file it allows all the browsers to be compatible with the code. Import method is not compatible with the older browsers but works with the newer browsers. However despite this fact it is still a rarely used method because the website will load up the website much slower than the link method. This is because the file is imported into the HTML file so the browser will initiate a download for the imported file before the content. This also applies to also mobile phones as the end-user will have missing content and errors whilst also having a slow load time. This could load up errors or have missing parts. This can be very frustrating for the end-user.

 

Both of these methods allows the user to put in multiple style sheets allowing there to be some structure and order, making it easier when troubleshooting. In actuality there is no real difference between the link and import method just that link is a newer and more updated version of import. These both allow the user to link the HTML file to an external sheet which allows the user to organise and carefully change anything quickly without changing the content information.
